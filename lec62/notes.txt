React provides us many functionalities 
Hooks --> Using the hook we can utilize the functionality
Hook is a utility using which we can hook on to the functionality provided by the React

We can use hooks in function based components

Every component can have a state with it.
Example :- Counter Component has a state count (that is the number of times the button is clicked) 

--> To manage a state we use useState Hook 
useState Hook provides 2 things --> 
    1. state Variable
    2. state Function --> Which is used to update the state
